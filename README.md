# PaySync Mobile App

**PaySync** is a mobile application built with **React Native** and **Expo** that serves as the frontend for a microservices-based finance system. The app allows users to manage payments, track financial transactions, and interact with backend financial services efficiently and securely.  

## Features

- User authentication and secure login/signup  
- View and manage transactions  
- Send and receive payments  
- Dashboard for financial analytics and summaries  
- Responsive design for both iOS and Android  
- Seamless integration with backend microservices  

## Technology Stack

- **Frontend:** React Native, Expo  
- **Backend:** Microservices (REST APIs)  
- **State Management:** Redux / Context API  
- **Navigation:** React Navigation  
- **API Requests:** Axios / Fetch  
- **Authentication:** JWT / OAuth
